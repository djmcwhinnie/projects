Scotle

Game Info

Welcome to another Wordle clone.

Based on the original game, with a twist to use only Scottish words and provide a meaning or description at the game's conclusion.

Resources + Inspiration

The game was built following a tutorial by Ania Kubow over on her YouTube channel, which you can find here.

The main intention was simply to practice JavaScript via the tutorial while building something I was interested in - between the game being on trend at the time, a variety of variations of the game popping up, and the idea to make a version of the game only using Scots words.

Once the main game was coded, I went about altering the code to change elements or add in some extra ideas, like the toggle for a light and dark mode. These small changes continued to stoke the intention of this side project as a means to practise and develop skills.

Future Development

With the core game built, I hope to continue improving and refining the site as an on going playground, with an idea to implement the following:

  • More Scots words.
  • Score Tracking (Wins v Losses).
  • Preventing possible random word duplication.
  • Refactoring responsive layout for keys and letter grids.
